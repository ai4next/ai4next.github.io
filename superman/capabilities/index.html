Capabilities

能力矩阵

多模型接入

支持 Gemini Vertex AI、OpenAI、DeepSeek、Claude、Ollama，以及 OpenAI-compatible API，可通过配置和环境变量切换。

文件与命令执行

提供 OS-aware shell 执行、文件读取、写入和精确 patch，适合代码库分析、自动修复、批处理和本地工作流。

持久会话

SQLite 保存完整消息，sessions 目录保存精简演化日志，并支持搜索、导出、导入、压缩、文件历史和 diff/revert。

专家委托

任务可委托给拥有独立系统提示、记忆和会话的专家 Agent，主 Agent 只聚合结果并推进最终目标。

分层自进化

agent evolver 从完成会话中优化主 Agent 与专家；meta evolver 只从 evolver 会话中改进进化过程本身。

即时通信入口

可作为常驻 server 接入 Telegram、飞书/Lark、企业微信、微信个人号、QQ、钉钉、Slack、Discord、LINE、微博等平台。

内建工具

Superman 默认提供 6 类内建工具，覆盖自治 Agent 在本地工作时最常见的执行、读写、修改和协作需求。

工具用途
exec根据当前操作系统使用 bash、sh 或 PowerShell 执行 shell 命令，并支持超时配置。
read按行读取文件，适合分析代码、配置、日志和文档。
write写入文件，受配置中的最大文件大小限制。
patch替换文件中的一个精确文本匹配，适合小范围、安全可控的修改。
ask在任务需要用户判断时中断并提问。
delegate当专家委托启用且存在专家时，动态加载并把任务交给专家子 Agent。

会话能力

持久会话是 Superman 与普通一次性 CLI Agent 的核心差异之一。它既保存完整消息，也保留更适合模型和进化系统读取的精简投影。

查询与回放

通过 sm sessions list、show、last、search 查看历史会话，理解 Agent 当时收到什么、做了什么、输出了什么。

文件历史

通过 files、history、diff、revert 查看会话中涉及的文件和修订，必要时回滚上一版本。

迁移与压缩

通过 export、import、compact 在不同环境间迁移会话或将旧上下文压缩为 summary。

存储维护

通过 storage 和 storage gc 查看持久会话存储统计并清理孤立文件快照。

模型与入口

模型层支持通过配置选择 provider、model name、base URL、API key 和自定义 headers。入口层则覆盖交互式 TUI、一次性 CLI、prompt 文件、自主反思模式和 IM server，便于把同一套 Agent 能力放进不同工作环境。

能力边界：Superman 的强项不是单点模型能力，而是把模型、工具、状态、专家和扩展机制组织成可长期维护的系统。
