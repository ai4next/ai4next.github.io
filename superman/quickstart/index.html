Quick Start

快速上手

macOS 或 Linux 可以直接安装最新发布版：

curl -fsSL https://raw.githubusercontent.com/ai4next/superman/main/install.sh | sh

Windows PowerShell：

iwr https://raw.githubusercontent.com/ai4next/superman/main/install.ps1 -useb | iex

初始化配置并启动：

sm init
export OPENAI_API_KEY=sk-...
sm

# 或执行一次性任务
sm run "这个目录里有什么？"

常用命令

命令说明
sm启动交互式终端聊天界面。
sm run "提示词"执行一次性任务并打印响应。
sm run -f prompt.txt从文件读取 prompt 执行。
sm configure查看或初始化配置。
sm toolsets列出当前配置的 ADK Skill 和 MCP toolsets。
sm reflect启动自主空闲监听和调度模式。
sm im serve启动即时通信接入 server。

最小配置

sm init 会从内嵌示例创建 config.yaml。最小情况下，你只需要确认模型 provider、模型名称、base URL 和 API key。

workspace: ${HOME}/.sm

model:
  provider: openai
  name: gpt-4o
  base_url: https://api.openai.com/v1
  api_key: ${OPENAI_API_KEY}

推荐的第一次验证

检查配置是否加载

运行 sm configure，确认配置文件路径和模型参数符合预期。

检查工具集合

运行 sm toolsets，确认内建工具、Skill 和 MCP server 是否按配置加载。

执行一个只读任务

先让 Superman 查看目录、总结 README 或解释配置，确认模型和文件读取正常。

查看会话记录

运行 sm sessions last 或 sm sessions list，确认会话已经被持久化。

从源码构建

如果需要本地开发或修改 Superman，可以从源码构建。项目需要 Go 1.26+。

go build -o sm .
./sm --help
